# Projet Ehura

## Task

 Build the back end for an e-commerce site by modifying starter code. Youâ€™ll configure a working Express.js API to use Sequelize to interact with a MySQL database.


## Usage

 - Clone the [repository](https://github.com/omgthegreenranger/project-ehura)
 - `npm install`
 - import the `./db/schema.sql` in to your MySQL database
 - `npm start`, stop server, `npm run seed`.
 - run server again
 - fire up a REST client such as Insomnia
 - send some stuff!
